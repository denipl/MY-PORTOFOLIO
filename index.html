Tentang Saya

Selamat datang di portofolio saya! Saya adalah seorang pengembang web dengan pengalaman dalam membangun aplikasi web yang efisien dan menarik. Saya bersemangat dalam menciptakan solusi digital yang tidak hanya fungsional tetapi juga memberikan pengalaman pengguna yang intuitif dan menyenangkan.

Keahlian saya mencakup pengembangan front-end dengan HTML, CSS, JavaScript (termasuk React dan Vue.js) serta pengembangan back-end menggunakan PHP (dengan Laravel) dan Node.js. Saya juga familiar dengan Python dan Go, serta menguasai Git untuk version control dan MySQL untuk manajemen basis data. Saya selalu antusias untuk terus belajar teknologi baru dan menghadapi tantangan pengembangan.
